How to Run the Project (Step-by-Step)

Using Jupyter Notebook
1 Install Dependencies
pip install -r requirements.txt

Typical libraries:

	- TensorFlow
	- NumPy
	- Pandas
	- OpenCV
 	- Matplotlib
	- Scikit-learn

2 Launch Jupyter Notebook
jupyter notebook

3 Open the Evaluation Notebook
notebooks/final_model_evaluation.ipynb

4 Configure Paths

Inside the notebook, update:

ROOT_DIR = "path_to_dataset"
WEIGHTS_PATH = "path_to_weights_file"

5 Run All Cells
Kernel → Restart & Run All
